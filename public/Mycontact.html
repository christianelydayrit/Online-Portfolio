MyContact - A C# and ASP.NET Windows Forms Application

MyContact is a project I built using C# and ASP.NET as a Windows Forms application. I developed this program as a way to practice C# and SQL, while also addressing a personal challengeâ€” I have a habit of misplacing or forgetting my contacts. This application serves as a simple yet efficient contact management system to help me organize and store important contact details.

Key Features:

  • Contact Management: The system allows users to record and store contacts in an SQL Server database. Each contact entry includes:
    • Contact ID
    • First Name
    • Last Name
    • Contact Number
    • Address
    • Gender
  • Add Functionality: Users can input new contact details through a form, and the system will save them into the database. The records are then displayed in a GridView, making it easy to track and view stored contacts.
  • Update & Delete: The application supports editing existing records and deleting contacts as needed for better data management.
  • Search Feature: A search function is integrated to help users quickly find a specific contact.
  • User-Friendly Interface: The application has a simple and intuitive UI, making it easy to navigate and use.

MyContact provides a practical solution for managing personal or professional contacts, offering essential CRUD functionality (Create, Read, Update, Delete) in a Windows Forms environment. This project has been a great way for me to enhance my skills in C#, SQL, and database management, while also building a tool that serves a real purpose.
