JokerApp - An ASP.NET Core Web Application

I created JokerApp to further expand my knowledge in different aspects of software development. This project serves as a practical exercise in ASP.NET Core, incorporating various fundamental web development principles. The application includes all the essential functionalities of a well-structured web page and is built using SQL, C#, and ASP.NET Core.

Technical Approach:

  • MVC Architecture: The application follows the Model-View-Controller (MVC) pattern, ensuring a structured and maintainable codebase. This approach helps separate concerns, making the application easier to develop and scale.
  • Entity Framework (EF) ORM: I took a different approach in database management by using Entity Framework (EF) as an Object-Relational Mapper (ORM). This allows the application to map SQL database tables to C# classes, simplifying database interactions and improving efficiency.
  • Razor Syntax: I utilized Razor, a templating language that combines C# and HTML, making it easier to dynamically render content while maintaining clean and readable code.

Key Features:

  • Add, Edit, Delete, and Search Jokes: Users can contribute to the joke database by adding, modifying, or removing jokes. A search function is available to quickly find specific jokes.
  • View Joke Details: Each joke entry has a dedicated details page where users can view full joke content.
  • Authentication & Authorization: Users are required to log in to access features like adding, editing, and deleting jokes. If they do not have an account, they can register directly within the app.

JokerApp is a fun and interactive way to practice and enhance my skills in ASP.NET Core, C#, SQL, and Entity Framework, while also providing users with a platform to share their cheesy and corny jokes. I hope you enjoy using this application as much as I enjoyed building it!
