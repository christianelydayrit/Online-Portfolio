My Online Resume

Here is the link to my personal online resume: Online Resume

This online resume was created to apply the knowledge gained from passing the AWS Certified Cloud Practitioner (CCP) exam. The webpage is hosted using an S3 bucket, and AWS CloudFront is used to enhance security, caching, and global accessibility. I also implemented JavaScript to count the visitors to the page, leveraging the combination of S3 and CloudFront to track and manage traffic effectively.

With this project, I demonstrated my ability to deploy static websites on AWS and use AWS CloudFront for fast content delivery worldwide. The visitor tracking implemented with JavaScript adds a dynamic feature to the otherwise static website.
