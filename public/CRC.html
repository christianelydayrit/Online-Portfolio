Online Portfolio

This Online Portfolio was built using a pre-made HTML5 template, which I customized and extended to reflect my personal background, technical skills, and project experience. The goal of the website is to present my most meaningful projects in a clear, structured, and accessible way.

I personalized the template by modifying the layout, styling, content structure, and media to better represent my development journey. The portfolio highlights projects across multiple domains, including web development, embedded systems, automation, and scripting.

The portfolio includes the following key features:

  • Project Showcase: A curated collection of standout projects, including full-stack web applications, Arduino-based systems, and Python projects developed for automation and problem-solving.
  • Interactive AI Assistant: The website integrates a personal AI assistant that visitors can interact with to ask questions about my background, skills, projects, and experience, providing a more engaging way to explore my profile.
  • Responsive Design: The customized HTML5 template ensures the website adapts properly across desktop and mobile devices.
  • Static Hosting & Deployment: The portfolio is hosted on AWS Amplify using static hosting, enabling fast delivery and reliable availability.

This portfolio serves as both a professional online presence and a technical showcase, demonstrating my ability to adapt existing tools, integrate interactive features, and present diverse projects in a cohesive and user-friendly format.
